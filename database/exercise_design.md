# 数据库设计练习题

1. 仿 [http://dict.cn/](http://dict.cn/) 设计一个简单的在线电子词典数据库表结构

2. 设计一个能够简单实现用户注册、登录、注销的数据库表结构

    ```
    用户注册
        邮箱： |________|
        帐号： |________|
        密码： |________|
        
        |注册|
    ```
    ```
    用户登录
        邮箱： |________|
        密码： |________|
     
        |登录|
    ```
    ```
    系统首页
    
        你好：xxx
        
        |注销|
    ```
    
3. 在题 2 的基础上，实现用户留言功能

    ```
    系统首页
        
        你好：xxx
        
        |注销|
        
        留言：+-----------------------------------—+
             |                                    |
             |                                    |
             |                                    |
             +-----------------------------------—+
        |提交|         
    ```
