# Online Examination

> 实现一个在线考试系统

基本功能
1. `系统管理员`
  - [ ] 登录
  - [ ] 维护管理员信息
2. `教师` 
  - [ ] 登录
  - [ ] 维护试卷信息
  - [ ] 判卷
3. `学生` 
  - [ ] 注册
  - [ ] 登录
  - [ ] 维护个人信息
  - [ ] 在线考试
4. `教务`
  - [ ] 登录
  - [ ] 维护班级信息
  - [ ] 维护学生信息
  - [ ] 管理考试信息
  - [ ] 查看考试结果

提高要求
 - [ ] 自动判卷
 - [ ] 报表打印
 - [ ] 防作弊
